Code Game Jame

Jeu réalisé en 24 h sur Unity

La 7e édition de la Code Game Jam a comme thème : maître du temps. Nous avions donc 24 h pour réaliser un jeu avec ce thème, en équipe de 6 ou 2 d'entre nous étaient en distanciel, dont moi-même.

Nous avons eu pas mal de temps pour réfléchir à une idée avant que les 24 h de code ne commencent, donc nous avons décidé de réaliser un "bullet hell" avec comme spécificité l'arrêt du "temps" ce qui fait que les balles s'arrêtent, s'accélèrent ou ralentissent grâce à notre pouvoir qui est d'arrêter le temps.

Mon rôle dans ce projet était développeur, j'ai donc travaillé en C# à l'aide d'Unity pour réaliser ce projet. J'ai découvert l'importance de définir des tâches à réaliser pour qu'il y ait un minimum d'organisation, et l'importance d'écrire un code de qualité et compréhensible à tous.
